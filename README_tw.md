# vlt-syslog-server (工作區)

專為 Windows 設計s的 Syslog 解決方案，致力於在多位元組字元 (CJK) 環境中提供極高的可視性與實用性。

## 專案結構與目標

本專案分為兩個主要元件，以滿足不同的使用場景。

### 1. [Portable](./Portable) (已發佈 v0.2.0)
- **狀態**: 已發佈。只要精力與體力允許，將持續維護。
- **理念**: 「裝在 USB 隨身碟裡的工程師多功能軍刀」。
- **特點**: 集成了 GUI 和伺服器引擎的單一執行檔。無需安裝，無需聯網，為現場除錯和臨時日誌收集提供卓越的靈活性。
- **初衷**: 作為一名忙碌s的工程師，我本著「如果有這個就方便了」的想法親手打造。希望能為同行們提供便利。

### 2. [Server](./Server) (開發中)
- **狀態**: 作為下一階段目標正在開發中。
- **理念**: 「Windows 伺服器上的專業級穩定運行」。
- **特點**: 將作為 Windows 服務在背景運行的「引擎」與僅在需要時連接的「監控前端」分離。
- **目標**: 為需要 24/7 全天候運行的生產環境提供堅固的日誌收集基礎設施。

---

## 開發背景

為了克服長期使用的 Windows 輕量級 Syslog 伺服器在「中日文顯示」方面的局限性，我使用 Rust 語言從零開始建構。該軟體具備「寬容解析 (Tolerant Parsing)」功能，能夠智能處理包括 Shift_JIS 以及帶或不帶 BOM 的 UTF-8 在內的各種實際編碼格式。

詳情請參閱各目錄下的 README。
