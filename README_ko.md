# vlt-syslog-server (워크스페이스)

2바이트 문자(CJK) 환경에서의 높은 가독성과 실용성을 추구한 Windows용 Syslog 솔루션입니다.

## 프로젝트의 구성 및 목적

본 프로젝트는 용도에 맞춰 두 가지 형태를 제공하기 위해 디렉토리를 분리하여 정리했습니다.

### 1. [Portable](./Portable) (v0.2.0 출시 완료)
- **상태**: 출시 완료. 체력과 기력이 닿는 한 유지보수를 지속할 예정입니다.
- **콘셉트**: "USB 메모리에 넣어 다니는 엔지니어의 만능 나이프".
- **특징**: GUI와 서버 엔진을 통합한 단일 실행 파일입니다. 설치와 인터넷 연결이 필요 없어, 현장 디버깅 및 일시적인 로그 수집 시 최고의 기동성을 제공합니다.
- **철학**: 바쁘게 일하는 엔지니어로서 "이런 게 있으면 좋겠다"는 생각으로 직접 만들었습니다. 비슷한 업무를 수행하시는 분들께서도 부담 없이 사용해 보시길 바랍니다.

### 2. [Server](./Server) (개발 중)
- **상태**: 차기 목표로 개발 중입니다.
- **콘셉트**: "Windows 서버를 위한 본격적인 안정 운용".
- **특징**: Windows 서비스로 백그라운드에서 동작하는 '엔진'과 필요할 때만 연결하는 '모니터링 프론트엔드'를 분리했습니다.
- **목적**: 상시 가동이 요구되는 운영 환경에서 견고한 로그 수집 기반을 제공합니다.

---

## 개발 배경

오랫동안 애용해 온 기존 Windows용 경량 Syslog 서버의 '일본어 표시 한계'를 극복하기 위해 Rust 언어로 처음부터 구축했습니다. Shift_JIS 및 BOM 유무에 관계없는 UTF-8 등 실제 현장의 다양한 인코딩을 지능적으로 판별하는 '관용적인 파シング(Tolerant Parsing)'을 특징으로 합니다.

상세 내용은 각 디렉토리의 README를 참조해 주세요.
