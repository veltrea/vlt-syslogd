# vlt-syslog-server (工作区)

专为 Windows 设计的 Syslog 解决方案，致力于在双字节字符 (CJK) 环境中提供极高的可视性与实用性。

## 项目结构与目标

本项目分为两个主要组件，以满足不同的使用场景。

### 1. [Portable](./Portable) (已发布 v0.2.0)
- **状态**: 已发布。只要精力和体力允许，将持续维护。
- **理念**: “装在 USB 闪存盘里的工程师多功能军刀”。
- **特点**: 集成了 GUI 和服务器引擎的单一可执行文件。无需安装，无需联网，为现场调试和临时日志收集提供卓越的灵活性。
- **初衷**: 作为一名忙碌的工程师，我本着“如果有这个就方便了”的想法亲手打造。希望能为同行们提供便利。

### 2. [Server](./Server) (开发中)
- **状态**: 作为下一阶段目标正在开发中。
- **理念**: “Windows 服务器上的专业级稳定运行”。
- **特点**: 将作为 Windows 服务在后台运行的“引擎”与仅在需要时连接的“监控前端”分离。
- **目标**: 为需要 24/7 全天候运行的生产环境提供坚固的日志收集基础设施。

---

## 开发背景

为了克服长期使用的 Windows 轻量级 Syslog 服务器在“中文/日文显示”方面的局限性，我使用 Rust 语言从零开始构建。该软件具备“宽容解析 (Tolerant Parsing)”功能，能够智能处理包括 Shift_JIS 以及带或不带 BOM 的 UTF-8 在内的各种实际编码格式。

详情请参阅各目录下的 README。
