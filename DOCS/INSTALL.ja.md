# vlt-syslogd — インストールガイド

English version: [INSTALL.md](INSTALL.md)

vlt-syslogd は syslog を受信して表示するソフトです。手順は OS ごとに分かれています。**自分の OS のページへ進んでください。**

| OS | インストールガイド |
|---|---|
| **macOS** | [INSTALL.macos.ja.md](INSTALL.macos.ja.md) |
| **Linux**（systemd） | [INSTALL.linux.ja.md](INSTALL.linux.ja.md) |
| **Windows** | [INSTALL.windows.ja.md](INSTALL.windows.ja.md) |

各ページは、その OS だけで完結する手順書です（冒頭で「Server / Console / Portable のどれを入れるか」を選べます）。

---

ソースから自分でビルドしたい場合は [BUILD.ja.md](BUILD.ja.md) を参照してください。

部品の構成・ポート・サービス識別子の一覧は [REFERENCE.ja.md](REFERENCE.ja.md) にまとめてあります（インストールには不要）。
